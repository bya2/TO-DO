유송수:
 -	개인적으로 권하는 것은 'DB sql을 이러한 논리로 이러이러하게 작성해보았다'라는 내용을 추가하는 것임.
 -	Table은 현재 런타임에서 데이터를 배열에 넣은 후, 연산해서 평균값을 구해 Table B에 넣고 있음. Table B(sec_show)에는 section 컬럼(구역 구분)과 sectionType 컬럼(구역 종류 구분)이 추가될 예정임. 따라서 Table A는 삭제 예정이므로 그림 수정을 권함.
 -	현재 웹 소켓이 아닌 SSE(Server-Sents Event) 방식을 사용 중임. (클라이언트와 서버 간 HTTP 연결 수립 후 HTTP 연결이 닫힐 때까지 서버에서 클라이언트로 데이터를 일방적으로 보내는 방법)
 - 	3.1에서 표는 한상현씨 표를 참고하여 쓰는 것을 권함.
 -	유송수씨는 이슈 관련해서 2개의 페이지를 알차게 쓰는 것이 좋다고 생각함.
