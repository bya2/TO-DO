```
결론부터 말하면, Rest와 Spread 문법은 정 반대의 개념이다.

Rest 파라미터는 함수에 전달된 인수들의 목록을 배열로 전달받기 위해 매개변수 이름 앞에 ...을 붙이는 것.

Spread 문법은 하나로 뭉쳐 있는 여러 값들의 집합을 펼쳐서 개별적인 값들의 목록으로 만드는 것.
```

## 스프레드 적용 대상

```
배열 결합(Array.prototype.concat),
배열 중간에 다른 배열의 요소들을 추가하거나 제거(Array.prototype.splice),
배열 복사(Array.prototype.slice),
이터러블을 배열로 변환, (이터러블이 아닌 유사 배열 객체는 제외)

```
