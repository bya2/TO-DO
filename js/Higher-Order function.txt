*Functional Programming (함수형 프로그래밍)

 함수를 다른 함수의 파라미터로 넘기거나 반환 값으로 함수를 받을 수 있는 프로그래밍 형태.


*First-Class function 

 함수는 Function Object
 함수를 Callback, Variable assign, .. 가능
 

*Higher-Order function (고차함수)

 함수를 인자로 받거나, 또는 함수를 반환함으로써 작동 하는 함수.
