OSI 7 Layer

    - OSI 7 계층은 독립적이지만 상하구조를 띄고 있음.
    - 따라서 윗 계층이 동작하려면 아랫 계층이 정상적으로 동작하고 있어야함.

    - 예를 들어, 네트워크 계층 이전에 아래 계층에서 어떤 작업을 하고 있기 때문에 IP 주소를 가지고 통신할 수 있음.

데이터 링크 계층:
    - 연결된 바로 이웃 컴퓨터와의 통신 만을 생각함.
    - 정의: 직접 연결된 서로 다른 2개의 네트워킹 장치 간에 데이터 전송을 담당하는 계층.
    - 이것을 보장해야 IP로 데이터를 보낼 수 있음.

    - 소프트웨어로부터 정보를 받으면 하드웨어로 연결해주는 연결다리.
    - 하드웨어로부터 데이터를 받아 우리가 받아서 볼 수 있도록 소프트웨어 특징을 갖는 LAYER로 전달.
