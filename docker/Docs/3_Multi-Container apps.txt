Multi Container apps

각 컨테이너들은 한 가지 일을 해야함.

1. 데이터베이스와 다르게, API나 프론트엔드는 확장해야할 가능성이 큼.
2. 별도의 컨테이너로 별도의 버전을 사용.
3. 데이터베이스 엔진와 애플리케이션을 별도로 배송.
4. 다중 프로세스 실행에는 '프로세스 관리자'가 필요하나, 컨테이너는 오직 하나의 프로세스로 시작.

~ 하나의 컨테이너가 다중 프로세스 실행을 하도록 해서, 프로세스 매니저가 필요하다면, 컨테이너 시작/종료 작업에 복잡성을 증가시킬 수 있음.

~ 시스템에서 다중 프로세스를 실행에는 '프로세스 관리자'가 필요.
